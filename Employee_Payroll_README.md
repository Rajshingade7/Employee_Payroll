# Employee Payroll

An employee payroll management system with full CRUD operations and a polished frontend interface. The application handles employee records, salary calculations, and payroll processing.

## Features

- Add, view, update, and delete employee records
- Salary and payroll computation
- Search and filter employees
- Clean frontend interface for easy management
- Data validation and error handling

## Tech Stack

- **JavaScript** — Core logic and interactivity
- **HTML5** — Page structure
- **CSS3** — Styling and layout

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Rajshingade7/Employee_Payroll.git
   cd Employee_Payroll
   ```

2. Open `index.html` in your browser, or use a local server:
   ```bash
   npx serve .
   ```
